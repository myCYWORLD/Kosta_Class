A안 (골프 플랫폼)

제목 : 골린이들을 위한 골프 플랫폼

사업목표 :  골프 초보들을 위한 정보 공유 플랫폼을 만들어 건전한 골프 문화 형성 

개발목표 :  Git 과 같은 형상 관리 툴을 통해 버젼을 관리하며, 
               팀원과의 원활한 소통을 위해 Notion, Slack과 같은 협업툴 사용
               SOLID 원칙을 준수하며 개발 진행

명단 : 박종우, 전승현, 조건우, 한미래, 황초연

서비스할 메뉴 : 
1. 비대면 코칭 서비스 (동영상)
2. 골프 의류 / 장비 판매 (장터)
3. 골프장 정보공유 커뮤니티
4. 경기 or 메이트 매칭 게시판
5. 너도나도 입으로는 쌉고수 자세코칭 게시판(동영상공유)

B안 (재고판매 플랫폼)

               합리적인 소비를 위한 서비스 제공 

개발목표 :  Git 과 같은 형상 관리 툴을 통해 버젼을 관리하며, 
               팀원과의 원활한 소통을 위해 Notion, Slack과 같은 협업툴 사용
               SOLID 원칙을 준수하며 개발 진행

명단 : 박종우, 전승현, 조건우, 한미래, 황초연

서비스할 메뉴 : 
1. 카테고리별 밀키트 재고 판/구매
2. 판매처 리뷰 
3. 재고 보유 판매처 지도 검색 기능
